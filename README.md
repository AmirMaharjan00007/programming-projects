## Programming Philosophy

My approach to programming has been shaped by both academic
study and real-world professional experience. During my IT
Systems Administrator internship at Addon Engineering Solutions
I saw firsthand how code quality, security, and documentation
directly impact the reliability and safety of systems that
real people depend on.

Key principles I apply:

**Security by design:**
Writing code with security considerations from the start —
input validation, proper authentication, avoiding hardcoded
credentials, and following OWASP guidelines for web applications.

**Readability and documentation:**
Code should be readable by others — meaningful variable names,
clear comments, and proper documentation.

**Problem solving:**
Programming is fundamentally about solving real problems. My
most meaningful programming work has been applied to real
contexts — including identifying and planning remediation of
security vulnerabilities in Angels Inc, our family retail
business.

## Cybersecurity Relevance

Programming knowledge is fundamental to cybersecurity practice:

- **Python** — most widely used language in cybersecurity tools,
  automation, and scripting
- **Web technologies** — understanding HTML, JavaScript, and PHP
  is essential for web application security testing
- **SQL** — SQL injection is one of the most common web
  application vulnerabilities — understanding SQL is essential
  to understanding and preventing it
- **Android development** — mobile security requires understanding
  of mobile application architecture
